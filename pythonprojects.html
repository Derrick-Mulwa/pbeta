Efficient Incremental Data Ingestion Pipeline

This project centers on developing an incremental data ingestion pipeline. It leverages efficient data handling techniques to ingest only new or updated records from the dataset, thus optimizing both processing time and system resource consumption. The project seamlessly integrates data retrieval from Kaggle, transformation via pandas, storage in MySQL, and additional backup to Amazon S3, making it scalable for large datasets.

Data Analysis of SimplyHired.com Job Postings

This project analyzes job postings for Data and Business Analysis positions using data scraped from SimplyHired.com. The goal is to explore the job market and test several hypotheses, such as whether higher-rated companies offer better salaries, whether jobs in bigger cities pay more, and how factors like job type, seniority, location, and benefits affect salaries and company ratings.

★ Hypothesis testing  ★ Data cleaning  ★ Data Analysis  ★ Python  ★ Jupyter Notebook   ★ Pandas  ★ Matplotlib
